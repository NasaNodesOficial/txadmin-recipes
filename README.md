# txadmin-recipes

## Recommended Resources
- [FiveM Disable Ambient Sounds](https://github.com/DikaN1337/fivem-disableambientsounds) (Standalone)
  - This script allow you to disable all the ambient sounds on FiveM.
- [No Exploding Pumps](https://github.com/MadsLeander/no_exploding_pumps) (Standalone)
  - This is a pretty simple recourse that removes the ability for pumps to blow up.
- [VehicleDeformation](https://github.com/Kiminaze/VehicleDeformation) (Standalone)
  - This is a small resource that allows saving and reapplying of vehicle bodywork deformation.
